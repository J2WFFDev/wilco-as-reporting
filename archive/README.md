# Archive

This folder preserves repository history that is no longer required for
production operation.

Archived files are reference-only:

- they are not imported by the Python package;
- they are not active GitHub Actions workflows;
- they should not be treated as current command documentation;
- current product documentation under `docs/` takes precedence.

## Folders

- `prompts/`: superseded development and Codex planning prompts.
- `docs/`: reserved for superseded documentation.
- `workflows/`: reserved for disabled legacy workflows.
- `experiments/`: reserved for retired experiments.
- `legacy/`: reserved for other historical implementation material.

## Current Archived Items

- `prompts/codex-master-prompt.md`: early development prompt superseded by the
  implemented CLI, current README, and product-specific documentation.
